## 💡 Quick Tips

### Add a New Page
1. Create component in `src/components/`
2. Copy glass-card pattern from existing components
3. Use classes from `global.css`
4. Add route to `src/App.js`
5. Test on mobile (f12) and desktop

### Customize Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: { 500: '#YOUR_COLOR' },
  accent: { 500: '#YOUR_COLOR' },
}
```

### Add Animation
In `src/styles/global.css`:
```css
@keyframes yourAnimation {
  from { ... }
  to { ... }
}
```

### Make Component
```jsx
<div className="glass-card p-8 rounded-3xl hover:glass-light smooth-transition animate-fadeInUp">
  Your content
</div>
```
